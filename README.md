# My-personal-helper-scripts
This is just a collection of scripts that make a few things easier for me

## pacfix.sh
This script repairs targeted package manager issues by removing corrupt download fragments, optimizing mirror infrastructure, reconstructing keyrings, and performing a full system sync.

## sys_audit.sh
This script performs a comprehensive system audit, collecting hardware, software, and repository information, and outputs the results to a timestamped text file.
